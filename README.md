# Glotonman.js

A game made with some html,css and javascript.

# How to play
Game site: https://seancasm.github.io/Glotonman.js/

- The main objective is to eat to stay away from 100/100 hungry points.
- **WASD** for directional movement.
- **P** to pause the game.
